# Translational Oncology submission checklist

Target journal: Translational Oncology

Recommended article type in Editorial Manager: Original Research Article

## Upload files

1. Manuscript source file: `manuscript.docx`
2. Title page: `title_page.docx`
3. Cover letter: `cover_letter.docx`
4. Highlights file: `highlights.docx`
5. Declaration statements: `declarations.docx`
6. Main figures as separate files: `Figure_1.png` to `Figure_5.png`
7. Supplementary material: `supplementary_material.docx`
8. Supplementary figure file, if requested separately: `Figure_S1.png`
9. Suggested reviewers, only if Editorial Manager asks: `suggested_reviewers.docx`

## Manuscript compliance

- Article type: Original Research Article.
- Structured abstract: 229 words, below the 250-word limit.
- Main text from Introduction through Conclusion: 4,376 words, below the 4,500-word limit.
- Main figures and tables: 5 figures + 1 table = 6 total.
- Supplementary items: 1 figure + 1 table.
- References: 24, below the 50-reference limit.
- Keywords: 7, within the 1 to 7 keyword limit.
- Highlights: 4 bullets, each 85 characters or fewer.
- Tables are editable text, not images.
- Data and code availability statement included.
- Declaration of competing interest included.
- Funding statement included.
- CRediT author contribution statement included.
- Generative AI disclosure included.

## Cover letter note

The guide requests an official institutional letterhead with logo. The supplied `cover_letter.docx` is formatted as an editable letter; use Taipei Medical University official letterhead if Editorial Manager or the journal office requires it.
